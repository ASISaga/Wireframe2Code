Bidirectional sync: JSON ↔ Figma nodes

Import — JSON → Figma

Choose a blueprint JSON file to create or update Figma nodes on the current page. Existing nodes are matched by their id and updated in place; new nodes are inserted; removed entries are deleted.

Export — Figma → JSON

Select a Figma node, then click Export. The plugin reads the node tree and downloads a blueprint JSON file you can commit alongside your schemas.
